{% trans "数据主权归还用户：本地优先的复兴" %}

{% trans "在云端统治一切的时代，我们选择退后一步，让计算发生在它最该发生的地方：您的设备。" %}

{% blocktrans %}当你在搜索引擎里随便找一个“JSON 格式化”工具并粘贴你的业务数据时，你是否考虑过：这段数据去了哪里？谁能看到它？它会被保存多久？在“本地优先 (Local-First)”的哲学中，这些担忧是不存在的。{% endblocktrans %}

1 {% trans "传统在线工具的陷阱" %}

{% blocktrans %}许多所谓的在线工具其实只是一个简单的网页表单，当你点击“处理”时，它们会将数据发送到远程服务器。这不仅带来了显著的网络延迟，更严重的是，你的商业机密或个人隐私完全暴露在了站长的面前。即便他们承诺不保存数据，你也无法验证这一点。{% endblocktrans %}

2 {% trans "本地优先：信任的物理基础" %}

{% blocktrans %}在 Ego Toolbox，我们拥抱“本地优先”的架构。对于 90% 以上的工具，我们的计算逻辑完全运行在您浏览器的 JavaScript 虚拟机中。数据流转的路径是：您的粘贴板 -> 浏览器内存 -> 浏览器处理 -> 您的屏幕。全过程没有一比特的数据流向互联网。这是基于物理层面的隐私保证。{% endblocktrans %}
