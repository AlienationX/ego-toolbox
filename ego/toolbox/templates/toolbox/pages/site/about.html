{% trans "我们的起源与初心" %}

{% blocktrans %}Ego Toolbox 的诞生源于一次日常开发中的挫败感。作为由多名全栈工程师和数据分析师组成的独立开发团队，我们在日常工作中频繁需要处理一些琐碎但关键的任务：格式化一段混乱的 JSON 响应、提取一张图片的 Base64 编码、或是将运营部门发来的异常 CSV 文件进行清洗。{% endblocktrans %}

{% blocktrans %}然而，我们在网络上寻找这些“免费工具”时，往往被迫忍受满屏的弹窗广告、伪装成下载按钮的恶意链接。这不仅打断了我们的心流（Flow），更带来了巨大的安全隐患。因此，我们决定自己动手打造一个理想中的在线工具库。Ego Toolbox 的第一原则是“干净与纯粹”。{% endblocktrans %}
