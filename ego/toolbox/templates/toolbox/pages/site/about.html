{% trans "关于我们" %}

{% trans "Ego Toolbox 的诞生是为了提供更简单、更快速、更纯粹的在线工具体验。" %}

{% blocktrans %}Ego Toolbox 诞生于对“干净、纯粹、高效”工具的渴望。在数字时代，我们每天都需要处理大量格式转换、数据整理和轻量创作任务，但很多在线工具让本来简单的事情变得嘈杂、复杂而不透明。我们希望提供另一种体验：更安静、更可靠，也更容易纳入日常工作流。{% endblocktrans %}

{% trans "1. 我们想解决的问题" %}

{% blocktrans %}很多工具确实能完成任务，但它们常常伴随着弹窗广告、杂乱界面、过度权限请求或不明确的数据处理方式。对高频使用场景来说，这些噪音会不断打断判断与执行。我们更关心的是，如何让一个十秒钟的任务真的在十秒钟内完成。{% endblocktrans %}

{% trans "2. 我们坚持的原则" %}

{% blocktrans %}我们相信好的工具应该“即用即走”。它不需要用户学习复杂操作，不需要夸张视觉来证明自己重要，也不应该试图夺走用户的注意力。对于大多数任务，合理的默认值应当足够好；界面的职责是减少犹豫，而不是制造选择成本。{% endblocktrans %}

{% trans "3. 隐私与边界" %}

{% blocktrans %}作为面向开发与效率场景的工具集合，我们非常重视数据边界。能够在浏览器本地完成的功能，我们优先采用本地处理；对于需要服务器资源的功能，我们尽量采用即时处理和最小化存留策略。用户数据不应成为产品增长的交换条件。{% endblocktrans %}

{% trans "4. 持续建设" %}

{% blocktrans %}Ego Toolbox 仍在持续演进。我们会继续扩展开发工具、数据处理工具和高频效率场景，也会不断优化页面性能、交互清晰度和整体一致性。如果你对某个工具有想法，或者希望我们优先支持某个场景，欢迎通过反馈页面告诉我们。{% endblocktrans %}
