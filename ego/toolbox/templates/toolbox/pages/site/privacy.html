{% trans "核心承诺：我们不收集您的数据" %}

{% blocktrans %}在 Ego Toolbox，我们认为最好的保护就是“不持有”。不同于大多数在线工具站，我们的技术架构是建立在“隐私由设计决定（Privacy by Design）”的原则之上的。我们致力于在不牺牲任何功能体验的前提下，最大限度地减少对用户数据的接触。{% endblocktrans %}

{% trans "数据处理机制" %}

{% trans "场景 A" %} {% trans "纯本地处理（Client-Side Only）" %}

{% blocktrans %}对于文本工具（JSON 格式化、Base64 转换、二维码解码等），所有的处理过程均在您的浏览器本地完成。这意味着您的数据**永远不会**离开您的计算机。我们甚至无法在服务器端看到您输入的内容。{% endblocktrans %}

{% trans "场景 B" %} {% trans "云端即时销毁（Ephemeral Processing）" %}

{% blocktrans %}对于必须在服务器上处理的复杂任务（如 AI 智能去背景），我们采用了加密传输和即时销毁技术。文件一旦处理完毕并返回结果给您，对应的临时文件将被系统即刻彻底覆写销毁（Hard Purge），绝不保留任何副本，也不用于任何机器学习训练。{% endblocktrans %}

{% trans "Cookie 与 访问日志" %}

{% blocktrans %}为了保障网站的正常运行、提供登录功能以及统计基础的访问流量，我们可能会使用少量的必要 Cookie 和匿名化的服务器访问日志。这些信息通常包括您的 IP 地址（我们会进行脱敏处理）、浏览器类型和访问页面。我们绝不使用广告追踪或第三方画像分析脚本（如 Facebook Pixel）。{% endblocktrans %}

{% trans "第三方服务" %}

{% blocktrans %}我们可能会在页面中嵌入少量的第三方统计脚本（如 Google Analytics）以优化用户体验，但我们已配置其自动匿名化 IP 信息。此外，对于广告合作伙伴（如 Google AdSense），他们可能会根据您的兴趣展示广告，但我们已选择停用所有个性化数据共享选项，以最大限度保护您的隐私。{% endblocktrans %}
