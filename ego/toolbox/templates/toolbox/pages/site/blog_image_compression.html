{% blocktrans %}一张图片占据了 Web 页面 60% 以上的带宽。如果我们能将图片体积缩小一半，页面的响应速度就会发生质的飞跃。从古老的 JPEG 到现在统治 Web 的 WebP，再到未来的霸主 AVIF，图像压缩技术正经历着一场静默而深刻的革命。{% endblocktrans %}

1 {% trans "WebP：谷歌送给 Web 的礼物" %}

{% blocktrans %}WebP 结合了预测编码（来自视频编码技术 VP8）和高级哈夫曼编码。在相同质量下，它的体积比 JPEG 小 30%，比 PNG 小 26%。更重要的是，它同时支持透明度和动画，是目前 Web 优化的首选格式。{% endblocktrans %}

2 {% trans "AVIF：极致压缩的新高度" %}

{% blocktrans %}AVIF 是基于 AV1 视频编码器的图像格式。它采用了更先进的块划分模型和帧内预测技术。虽然目前编码速度较慢，但其极致的压缩比让它成为了流量大户（如 Netflix）的宠儿。{% endblocktrans %}
