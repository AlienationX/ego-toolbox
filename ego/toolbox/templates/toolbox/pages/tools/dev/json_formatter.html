{% trans "关于 JSON 格式化的技术科普" %}

{% trans "什么是 JSON？" %}

{% blocktrans %}JSON（JavaScript Object Notation）是一种轻量级的数据交换格式。它基于 ECMAScript 的一个子集，采用完全独立于编程语言的文本格式来存储和表示数据。由于其天生具有结构清晰、层级分明且极大程度降低了信息冗余（相比于 XML）的特点，JSON 已经成为了现代 Web API 和微服务前后端通信的绝对标准格式。{% endblocktrans %}

{% trans "我们为什么需要 JSON 格式化工具？" %}

{% blocktrans %}在实际的代码开发与抓包调试中，服务端返回的 JSON 数据通常被“压缩（Minified）”成了没有空格和换行的单行长字符串，以此来节约网络传输带宽。但这对于人类开发者来说是极其难以阅读和排错的。利用我们的 JSON 格式化解析器，不仅能够瞬间将其转换为结构化的缩进树，还能帮助您定位缺少括号、引号错位等常见的语法 Syntax Error。{% endblocktrans %}

{% trans "纯本地处理，拒绝数据上传" %}

{% blocktrans %}很多在线 JSON 工具会将您提交的文本发送到后台服务器进行解析，这存在极大的安全漏洞——因为 JSON 经常包含诸如 Access Token、数据库连接串、或者是包含用户隐私的业务报文。Ego Toolbox 的 JSON 解析器 100% 运行在您的浏览器本地（利用原生的 JavaScript 引擎处理）。当您关闭页面时，所有数据灰飞烟灭，绝无拦截与记录。{% endblocktrans %}

{% trans "JSON 压缩 (Minify) 的使用场景" %}

{% blocktrans %}除了格式化（Beautify）使其适于阅读外，本工具提供的“压缩”功能同样重要。当您需要将静态的 JSON 配置项硬编码写入数据库字段（如 MySQL 的 LongText 或 JSON 列），或者将参数填入 POSTman / Curl 终端时，去除所有的换行和空格能够有效避免不可见控制字符引发的解析错误。{% endblocktrans %}
